# ClipGuardian — 剪辑守护

> 实时监控 Premiere Pro / After Effects / 剪映专业版运行状态，在剪辑软件工作期间自动定时备份项目文件夹，防止崩溃导致内容丢失。

## 功能列表

- **进程监控**：实时检测 Adobe Premiere Pro、After Effects、剪映专业版是否正在运行，可自由勾选监控目标
- **自动定时备份**：检测到目标软件运行时，按设定间隔自动将监控文件夹打包为 ZIP 压缩备份
- **增量备份**：基于文件夹内容哈希（SHA256）判重，仅在文件发生变化时才创建新备份，避免重复备份浪费空间
- **备份历史管理**：完整记录所有备份的时间、大小、来源，支持双击恢复备份和删除旧备份
- **自动清理**：每个文件夹最多保留 50 个备份，超出后自动删除最旧的备份
- **手动即时备份**：支持不依赖守护进程的「立即备份」按钮，随时手动触发
- **配置持久化**：所有设置（监控软件、文件夹、备份间隔、备份路径）自动保存到 `~/.clip_guardian/config.json`

## 安装方法

### 环境要求

- Python 3.8+
- psutil（进程检测）

### 快速安装

```bash
# 方式一：一键安装脚本（Windows）
install.bat

# 方式二：手动安装
git clone git@github.com:solcat1007/clip-guardian.git
cd clip-guardian
pip install -r requirements.txt
```

### 依赖说明

| 依赖 | 版本要求 | 用途 |
|---|---|---|
| psutil | 最新即可 | 跨平台进程检测 |
| 其余全部 | — | Python 标准库（tkinter / zipfile / hashlib / json / threading） |

## 详细使用

### 启动

```bash
python clip_guardian.py
```

### 操作步骤

1. **软件监控**：勾选需要监控的剪辑软件（PR / AE / 剪映），至少勾选一个
2. **添加监控文件夹**：输入或浏览选择需要备份的项目文件夹，点击「＋ 添加」加入监控列表
3. **设置备份间隔**：在下拉框中选择备份间隔（30秒 / 1分钟 / 2分钟 / 5分钟）
4. **设置备份位置**：选择备份 ZIP 文件的存储目录（默认 `D:\ClipBackups`）
5. **点击「▶ 开始守护」**：工具开始循环扫描进程，检测到任一目标软件运行后开始定时备份
6. 守护期间可点击 **「⚡ 立即备份」** 手动触发一次备份
7. 关闭窗口或点击 **「■ 停止守护」** 停止守护

### 备份文件结构

```
备份目录/
├── 项目文件夹A/
│   ├── 项目文件夹A_20260708_143022.zip
│   ├── 项目文件夹A_20260708_143522.zip
│   └── …（最多 50 个）
├── 项目文件夹B/
│   └── 项目文件夹B_20260708_143022.zip
└── ...
```

### 恢复备份

在备份历史列表中**双击**某条记录，弹出确认对话框后选择恢复目标文件夹，即可将 ZIP 内容解压还原。

## 输入输出说明

### 监控目标进程

| 软件 | 进程名 |
|---|---|
| Adobe Premiere Pro | `Adobe Premiere Pro.exe` |
| Adobe After Effects | `AfterFX.exe` |
| 剪映专业版 | `JianyingPro.exe` |

### 备份格式

- 压缩格式：ZIP（deflate 压缩）
- 备份判重：基于文件夹内所有文件路径 + 文件大小 + 修改时间的 SHA256 哈希，内容无变化则跳过
- 备份包含：监控文件夹内的全部文件（排除 `.git` / `__pycache__` / `.clip_backups` 目录）

## 注意事项

1. 自动备份仅在检测到目标剪辑软件**正在运行**时才会触发
2. 备份间隔支持 30秒 / 1分钟 / 2分钟 / 5分钟，过短间隔会产生大量备份，建议根据项目修改频率选择合适间隔
3. 每个文件夹最多保留 50 个备份，超出自动删除最旧的
4. 恢复备份将**覆盖**目标文件夹中的同名文件，操作前会弹窗确认
5. 剪映仅支持**剪映专业版**（桌面版），不支持移动版或网页版，进程名为 `JianyingPro.exe`

## 技术栈

| 层级 | 技术 |
|---|---|
| GUI 框架 | Tkinter（Python 标准库） |
| 进程检测 | psutil |
| 压缩备份 | zipfile（Python 标准库，ZIP_DEFLATED） |
| 哈希判重 | hashlib.sha256 |
| 运行环境 | Python 3.8+ |
| 打包建议 | 可用 PyInstaller 打包为独立 .exe |
